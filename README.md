# 🛠️ Utility Box - Caixa de Ferramentas

## 📋 Descrição

O **Utility Box** é uma aplicação web completa que reúne múltiplas ferramentas úteis em um só lugar. Desenvolvido com tecnologias modernas, oferece uma experiência de usuário excepcional com interface responsiva e funcionalidades avançadas.

## ✨ Ferramentas Disponíveis

### 📝 **Bloco de Notas**
- Editor de texto completo com auto-save
- Sistema de organização por notas
- Busca e filtros
- Exportação/importação de dados
- Atalhos de teclado (Ctrl+S, Ctrl+N)

### 🧮 **Calculadora**
- Operações básicas (adição, subtração, multiplicação, divisão)
- Funções científicas (raiz quadrada, potência, inverso)
- Histórico de cálculos
- Suporte a teclado
- Formatação automática de números

### 🗂️ **Sistema de Post-its**
- ✅ Criação de lembretes coloridos (6 cores disponíveis)
- ✅ Múltiplas cores disponíveis
- ✅ Organização por data
- ✅ Edição e exclusão
- ✅ Interface drag & drop
- ✅ Exportação/importação de dados
- ✅ Sistema de categorias

### 📊 **Organizador de Tarefas**
- ✅ Sistema Kanban (4 colunas: Pendente, Em Progresso, Concluída, Cancelada)
- ✅ Prioridades configuráveis (Baixa, Média, Alta, Urgente)
- ✅ Categorias (Trabalho, Pessoal, Estudo, Saúde, Finanças, Outros)
- ✅ Gráficos estatísticos com Chart.js
- ✅ Drag & drop entre colunas
- ✅ Sistema de datas de vencimento
- ✅ Exportação/importação de dados

### 🃏 **Jogo de Paciência**
- ✅ Solitário clássico com regras completas
- ✅ Sistema de pontuação e estatísticas
- ✅ Histórico de jogos
- ✅ Sistema de dicas
- ✅ Interface responsiva
- ✅ Atalhos de teclado
- ✅ Persistência de estatísticas

### 💻 **Editor de Código**
- ✅ Suporte a 15+ linguagens de programação
- ✅ Múltiplos temas (Padrão, Escuro, Monokai, Dracula, GitHub, VS Code)
- ✅ Tamanhos de fonte configuráveis
- ✅ Múltiplas abas de edição
- ✅ Sistema de arquivos com extensões automáticas
- ✅ Exportação e importação de arquivos
- ✅ Atalhos de teclado (Ctrl+N, Ctrl+S, Ctrl+O)
- ✅ Detecção automática de linguagem por extensão

## 🛠️ Tecnologias Utilizadas

- **Frontend**: HTML5, CSS3, JavaScript ES6+
- **Framework CSS**: Bootstrap 5.3.0
- **Ícones**: Bootstrap Icons 1.11.0
- **Gráficos**: Chart.js
- **Armazenamento**: localStorage/IndexedDB
- **Responsividade**: CSS Grid + Flexbox

### ⌨️ **Sistema Completo de Atalhos de Teclado**

#### 🌐 **Atalhos Globais**
- **Ctrl/Cmd + 1-6**: Navegar entre abas
- **Esc**: Fechar modais
- **F1**: Mostrar ajuda completa
- **Ctrl/Cmd + /**: Lista completa de atalhos
- **Ctrl/Cmd + T**: Alternar tema claro/escuro
- **Ctrl/Cmd + Shift + S**: Sincronização manual
- **Ctrl/Cmd + Shift + E**: Exportar dados
- **Ctrl/Cmd + Shift + I**: Importar dados

#### 📝 **Blocos de Notas**
- **Ctrl/Cmd + S**: Salvar nota atual
- **Ctrl/Cmd + N**: Nova nota
- **Ctrl/Cmd + D**: Duplicar nota selecionada
- **Ctrl/Cmd + F**: Buscar em notas
- **Delete**: Deletar nota selecionada
- **Enter**: Salvar (no campo título)
- **Ctrl/Cmd + Enter**: Nova nota

#### 🧮 **Calculadora**
- **Enter/=**: Calcular resultado
- **Escape**: Limpar calculadora
- **Backspace**: Apagar último caractere
- **Delete**: Limpar calculadora
- **C**: Limpar (atalho rápido)
- **Ctrl/Cmd + H**: Mostrar/ocultar histórico

#### 📋 **Post-its**
- **Ctrl/Cmd + N**: Novo Post-it
- **Ctrl/Cmd + S**: Salvar Post-it
- **Ctrl/Cmd + E**: Editar Post-it selecionado
- **Ctrl/Cmd + C**: Copiar Post-it para área de transferência
- **Enter**: Salvar (quando editando)
- **Escape**: Cancelar edição

#### ✅ **Organizador de Tarefas**
- **Ctrl/Cmd + T**: Nova tarefa
- **Ctrl/Cmd + S**: Salvar tarefa
- **Ctrl/Cmd + F**: Buscar tarefas
- **Ctrl/Cmd + Shift + C**: Filtrar por categoria
- **Enter**: Salvar (no modal)
- **Escape**: Fechar modal

#### 🃏 **Jogo de Paciência**
- **Ctrl/Cmd + N**: Novo jogo
- **H**: Mostrar dica
- **Espaço**: Comprar carta
- **Ctrl/Cmd + Z**: Desfazer movimento
- **Ctrl/Cmd + R**: Reiniciar jogo

#### 💻 **Editor de Código**
- **Ctrl/Cmd + N**: Novo arquivo
- **Ctrl/Cmd + S**: Salvar arquivo
- **Ctrl/Cmd + O**: Abrir arquivo
- **Ctrl/Cmd + F**: Buscar no código
- **Ctrl/Cmd + H**: Substituir no código
- **Ctrl/Cmd + W**: Fechar aba atual
- **Ctrl/Cmd + Tab**: Próxima aba
- **Ctrl/Cmd + Shift + Tab**: Aba anterior
- **F1**: Ajuda do editor

#### 🔧 **Controles do Sistema**
- **Botão de Toggle**: Ativar/desativar todos os atalhos
- **Indicador Visual**: Status dos atalhos na interface
- **Ajuda Contextual**: F1 mostra ajuda específica da ferramenta ativa
- **Prevenção de Conflitos**: Atalhos são desabilitados em campos de texto quando necessário

## 🚀 Como Executar

### **Requisitos**
- Navegador moderno (Chrome 90+, Firefox 88+, Safari 14+, Edge 90+)
- JavaScript habilitado
- Conexão com internet (para CDNs)

### **Instalação**
1. **Clone ou baixe** os arquivos do projeto
2. **Abra** o arquivo `index.html` em um navegador
3. **Navegue** pelas ferramentas usando as abas
4. **Teste** todas as funcionalidades

### **Desenvolvimento**
```bash
# Estrutura de pastas
mkdir utility-box
cd utility-box

# Criar arquivos
touch index.html
mkdir css js
touch css/main.css css/tools.css css/responsive.css
touch js/main.js js/notes.js js/calculator.js js/postits.js js/tasks.js js/solitaire.js js/auth.js
```
